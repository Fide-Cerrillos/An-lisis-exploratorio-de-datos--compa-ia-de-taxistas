# Análisis Exploratorio de Datos: Compañías de Taxis en Chicago

Este proyecto fue realizado como parte del diplomado en Ciencia de Datos de **Tripleten**.

## Objetivo
Analizar la estructura del mercado de taxis para identificar líderes y patrones de comportamiento en los barrios de Chicago.

## Conclusiones clave
* Se identificó una estructura de mercado oligopólica.
* Alta concentración de viajes en barrios específicos.

## Tecnologías usadas
* Python (Pandas, Matplotlib, Seaborn)
* Jupyter Notebook
